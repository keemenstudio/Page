Edmonton House Price Prediction

Data Cleaning: There are over 40000 Edmonton house price data with 26 features. Drop the missing columns and replaces the categorical data with numeric numbers.

Exploratory Data Analysis(EDA): Plot the correlation matrix and price-location heat map, and join the missing data to find that training data split into two main categories: with the built house or without.

Models: Linear regression, Lasso, Support Vector Regressor, Decision tree, Random forest, Gradient boosting, Multi-layer perceptron (MLP), Radial basis neural network (RBFN)

Among all the models, the gradient boosting regressor performs best and then makes the backward feature selection with 10-fold cross-validation.

Mean Absolute Persentage Error for (MAPE) gradient boosting regressor is 4.46%, improved 49% from the baseline model (linear regression, MAPE = 8.88%).
