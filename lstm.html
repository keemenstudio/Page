Multiple Time Series Prediction

In the train data that I cleaned, there are 3,000,887 numbers of data that contains 54 different stores with 33 different products group and their sales data in the time range from 2013-01-01 to 2017-08-15. For each day, there are 54 * 33 equals 1782 time series sales data.

The architecture of the LSTM model is designed in three main layers, each layer contains one LSTM layer, one batch normalization layer and one dropout layer. Finally, add a time-distributed layer.

LSTM successfully predict the future 16 days sales data with acceptable errors. Evaluation uses Root Mean Squared Logarithmic Error (RMSLE)

Empty Entry: 4.3796

BaseLine: 0.73163

ARIMA: 0.56345

LSTM: 0.47928
